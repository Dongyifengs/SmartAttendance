# 常用 Git 提交 Emoji 图标参考

| Emoji | 代码                       | 使用场景      | 实例提交消息                         |
|-------|--------------------------|-----------|--------------------------------|
| ✨     | `:sparkles:`             | 新功能       | `✨ Add user login feature`     |
| 🐛    | `:bug:`                  | 修复 Bug    | `🐛 Fix login page crash`      |
| 📚    | `:books:`                | 文档更新      | `📚 Update API documentation`  |
| 🎨    | `:art:`                  | 代码样式/结构优化 | `🎨 Improve code formatting`   |
| ⚡     | `:zap:`                  | 性能优化      | `⚡ Optimize database query`    |
| 🔥    | `:fire:`                 | 删除代码/文件   | `🔥 Remove deprecated lib`     |
| 💄    | `:lipstick:`             | UI 样式更新   | `💄 Update button styles`      |
| 🚀    | `:rocket:`               | 部署相关      | `🚀 Deploy v2.0 to production` |
| 🔧    | `:wrench:`               | 配置文件修改    | `🔧 Update webpack config`     |
| ♻️    | `:recycle:`              | 代码重构      | `♻️ Refactor auth module`      |
| 🗑️   | `:wastebasket:`          | 清理废弃代码    | `🗑️ Clean up unused utils`    |
| 👥    | `:busts_in_silhouette:`  | 贡献者相关     | `👥 Add project author`        |
| 🚧    | `:construction:`         | 进行中的工作    | `🚧 WIP: Payment system`       |
| ✅     | `:white_check_mark:`     | 测试通过      | `✅ Pass all unit tests`        |
| 🔒    | `:lock:`                 | 安全相关      | `🔒 Fix XSS vulnerability`     |
| 🌐    | `:globe_with_meridians:` | 国际化       | `🌐 Add i18n support`          |
